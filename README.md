# Context-Todo-App

Context-Todo-App is a functional React application that manages todos, filters, and theme state using the Context API. It avoids prop drilling and external libraries by structuring multiple independent contexts for scalable and maintainable state management.

## Features

- Add, complete, and delete todos
- Filter todos by status (All, Active, Completed)
- Toggle light/dark theme
- Multiple Context Providers for todos, filters, and theme
- Persistent state using `localStorage`
- Optimized updates with `useReducer` and `useContext`

## Tech Stack

- React
- TypeScript
- Context API
- LocalStorage API

---

This app provides hands-on experience with scalable state architecture using React Context.
